FripMarket

FripMarket est une plateforme web de vente et d'achat de vêtements et accessoires de seconde main. Le site permet aux utilisateurs de vendre, acheter et découvrir des articles uniques tout en favorisant une consommation responsable et durable.

Fonctionnalités

Catalogue de produits : Parcourir les articles disponibles avec filtres par catégorie, taille, prix, et popularité.

Gestion des annonces : Les vendeurs peuvent créer, modifier ou supprimer leurs annonces facilement.

Recherche avancée : Trouver rapidement des articles spécifiques grâce à des filtres intelligents.

Panier et commandes : Ajouter des articles au panier et suivre les commandes.

Profil utilisateur : Gérer son profil, ses annonces et ses commandes.

Messagerie intégrée : Communiquer entre acheteurs et vendeurs directement sur la plateforme.

Technologies utilisées

Frontend : React.js, CSS/SCSS, HTML5

Backend : Node.js, Express.js

Base de données : MongoDB

Authentification : JWT / OAuth

Déploiement : Vercel / Heroku / Render (selon configuration)
